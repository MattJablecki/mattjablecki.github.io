On Display
Python(Tkinter/sqlite3)

Created for a Tech Academy assignment, this project, developed using Python, allows the user to choose a source directory and a destination directory from their computer and pass files from one to the other. What's more, when the files are passed, the names of the files and their most recent modification date and time are printed to the console, as well as stored in a database.

By leveraging the Tkinter GUI package in conjunction with the sqlite3 module, I was able to create a fully functional desktop app that allows for files to be moved with ease, as well as keep a detailed record of which files have been moved.
